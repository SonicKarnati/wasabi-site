# Design-X Creative Direction

## 1. Concept

A cinematic, scroll-driven journey through future-ready learning in India: moving from raw potential, to hands-on creation, to visible proof, to an invitation to build with Wasabi.

This is intentionally more intense than the stable institutional website. It should feel like a high-end interactive demo reel while still preserving the core Wasabi story: people become capable by creating.

## 2. Visual Language

- Earthy cream base.
- Forest green institutional grounding.
- Terracotta warmth used as motion/accent energy.
- Deep charcoal text.
- Electric-feeling highlights created through motion and glow, not neon.
- Documentary images.
- Animated UI overlays.
- Projector-light atmosphere.
- Kinetic typography.
- Scroll-driven layered panels.
- Soft grain and grid texture.
- Subtle glass only for floating signal cards.
- High-contrast editorial sections.

## 3. Motion Language

- Fast but intentional.
- Masked headline reveals.
- Cursor-reactive glow.
- Kinetic marquee transitions.
- Sticky split-layout storytelling.
- Parallax-style floating signal cards.
- Hover-reactive program cards.
- Animated borders.
- SVG line-drawing system map.
- Magnetic CTA buttons.
- Reduced-motion fallback for all major motion.

## 4. Website Story Arc

1. Cinematic hero: future-ready skills, real projects, India's next builders.
2. Mission kinetic transition: Learn. Build. Evolve. Human capability.
3. Problem / future gap: access alone is not capability.
4. Hands-on learning: the first real build changes confidence.
5. Programs as animated cards: program breadth with visible claim status.
6. Student/project proof: placeholders stay visible.
7. Impact / future horizon: proof must be honest.
8. Contact / partnership CTA: invite serious conversation without overpromising.

## 5. Asset Plan

Priority assets:

1. Hero classroom still or motion loop.
2. Kinetic typography mission transition.
3. Project-build montage.
4. Abstract curriculum system map.
5. Final CTA cinematic loop.

This pass generated the hero still only. The remaining media slots are represented through CSS/SVG/frontend motion and documented as future Higgsfield candidates.

## 6. Risk List

- Too much animation.
- Slow load time.
- Motion sickness.
- Generic AI aesthetic.
- Inconsistent generated people/assets.
- Unreadable text.
- Unsupported credibility claims.
- Mobile breakage.
- Free-plan credit limits.
- Generated image rights/approval ambiguity.
