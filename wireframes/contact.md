# Contact Page Wireframe

Purpose: Route different visitor types to the right inquiry path while official contact details are still being confirmed.

Audience: Government, schools, parents, students, institutions, partners, media, and potential supporters.

Primary CTA: Send Inquiry

Secondary CTA: Choose Inquiry Type

Review questions:

- What is the official public domain/email/phone?
- Which inquiry types should appear in version 1?
- Should student/parent enrollment be separate from institutional partnership?
- What privacy or response expectation text is required?
