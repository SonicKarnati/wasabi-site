# Programs / Work Page Wireframe

Purpose: Show what Wasabi actually does across student, school, SHG, public-sector, startup, and event contexts.

Audience: Government visitors, schools, parents, institutional partners, SHG ecosystem stakeholders, startup builders, and students.

Primary CTA: Discuss a Program

Secondary CTA: Ask About Enrollment

Review questions:

- Are these the right program categories for version 1?
- Which programs are active, completed, proposed, or campaign concepts?
- Does each program show audience, format, and outputs clearly?
- Are partner-sensitive claims handled carefully?
